Randomized Quiz App

A full-stack quiz application built using HTML, CSS, React, JavaScript, MySQL, and XAMPP. The application is designed with a responsive interface for smooth usage across devices.

Technologies: HTML, CSS, React, JavaScript, MySQL, XAMPP

Blockchain-based Supply Chain Management

A decentralized agriculture supply chain management system implemented using Remix IDE and Ethereum Blockchain. The system ensures transparency, traceability, and efficiency by recording each stage of the supply chain on the blockchain. Smart contracts handle crop tracking, shipment status, and inventory management for farmers, distributors, retailers, and customers.

Form: Blockchain Application
HTML: HTML5
Used Languages: Solidity, JavaScript
Technologies: Remix IDE, Ethereum, MetaMask, Smart Contracts

URL Shortener Web App

A web application that shortens long URLs into easy-to-share links. Built using HTML, CSS, JavaScript, Node.js, and MongoDB, it includes user authentication, analytics for tracking link clicks, and a clean, responsive interface. The backend API efficiently maps short URLs to their original targets for quick redirection.

Form: Web Application
HTML: HTML5
Used Languages: JavaScript
Technologies: HTML, CSS, JavaScript, Node.js, MongoDB, Express.js
